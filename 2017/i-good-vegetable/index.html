如果你想在数组里处理一段有相同性质的元素（比如权值）
注意循环会不会处理第一个元素。
比较好的写法是先声明一个起始坐标 ‘int beg = i’
然后’while (i <= n && w[i] == w[beg]) {
//something
i++
}’
注意i结束时是超尾的，所以要么循环结束之后立刻i—，要么把外层循环的i增去掉

还有一种写法

for (int i = 1, j; i <= n; i = j) {
    for (j = i; j <= n && w[j] == w[i]; j++) {
      //something
    }
}

看起来还是第二种比较好
