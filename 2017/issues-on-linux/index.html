文档

PDF 特殊中文字体无法显示

Okular 使用 poppler 进行 PDF 渲染。poppler 有一个可选依赖 poppler-data 支持 CJK 渲染。安装即可。

zip 压缩文件内中文文件名乱码

zip 文件格式本身不储存编码信息，导致默认编码设置为 UTF-8 的 Linux 无法正常处理 Windows 下的压缩文件。
Ubuntu 似乎对 zip 打了个 patch 来支持。
Arch 可以选择安装 aur 中的 *-natspec 系列包。
如果你使用 ark，可以装 unzip-natspec 还是 p7zip-natspec 来着。反正一般是要都装的……装完就好了。
