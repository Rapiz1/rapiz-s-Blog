为通过 SSH 的 Git 配置代理

查了一大堆，说什么方法的都有……
By Rapiz 2020-02-21

一开始在~/.ssh/config里加这个不管用。

Host github.com
    User                    git
    ProxyCommand            nc -x 127.0.0.1:1080 %h %p

一直是啥密匙交换错误，无法解析主机啥的。
我一开始还以为是加密问题。
后来仔细一看，不对劲啊 can't resolve host 127.0.0.1:1080
这主机都解析不了还指望解析啥……
然后又查了查，发现ubuntu的netcat不支持-x。但我寻思我用 arch啊。man了一下发现果然没有这个选项……
pacman -Ss netcat发现有openbsd-netcat。换掉gnu-netcat，解决。
