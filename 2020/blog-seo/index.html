Sitemap

首先加 Sitemap。Sitemap 类似于网站的一个地图，蜘蛛会定期根据这个地图遍历网站。也许大概可能百度会根据这个收录本博的文章。

这篇文章 讲了怎么用 Hexo 生成 google 和百度的 sitemap。

你可以访问 https://rapiz.me/sitemap.xml https://rapiz.me/baidusitemap.xml 来查看本站的 google sitemap 和百度 sitemap。

这里描述了 Hexo 中通过 API 向百度提交。

Metadata

然后我看了下别人的网站，<head> 里都有 <meta> 表明关键字和作者啥的。我觉得可能 <meta> 多了也利于收录？这篇文章里有挺多常用的标签，我加了 author, description, keywords，希望多少起点作用吧。

外链

<a> 标签的目标页面会分享页面权重。除了Hexo 官方提供了一个插件 会给所有出站链接加上 no follow 等属性，防止搜索引擎蜘蛛爬出站点，降低博客文章权重。

注意把友链在插件设置里排除。我们希望通过友链和朋友交换权重。
