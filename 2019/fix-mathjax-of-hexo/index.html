修复 Hexo 的 Mathjax 支持

In DEV By Rapiz 2019-09-08 Hexo

之前一直用 hexo-renderer-mathjax 插件渲染数学公式，最近发现它失效了。

找到 Github 项目发现已经很久没更新了，似乎由于内置的 CDN 失效而渲染失败。

然后在 Issue 里找到了 Fork hexo-render-mathjax2

这个版本可以自定义 CDN 地址。换用 bootcss 的 CDN 即可解决问题。
